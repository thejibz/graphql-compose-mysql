# graphql-compose-boilerplate

## Includes

- Babel (ES6, babel-preset-env)
- ESLint
- Flowtype
- express
- express-graphql
- graphql
- graphql-compose
- nodemon

## Usage

```bash
git clone https://github.com/graphql-compose/graphql-compose-boilerplate

cd graphql-compose-boilerplate

# make it to your own
rm -rf .git

yarn install

# start server with reloading on file changes
yarn dev

# OR start server
yarn start
```
